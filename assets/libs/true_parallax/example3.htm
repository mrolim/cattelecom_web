Changing the properties

Below you will find working example(s) and complete, ready to use HTML code for each example.

Remember: always include the correct path to jQuery library and the parallax plugin:

 <!-- PROVIDE THE CORRECT PATH TO .js FILES -->
<script type="text/javascript" src="/path/to/jquery-latest.js"></script> 
<script type="text/javascript" src="/path/to/jquery.real-parallax.js"></script>

Applying different properties

Parallax has many options you can pass in at initialization to achieve different effects.

Z=70

Z=0

Z=-70

sensitivity X ( )
sensitivity Y ( )
easingCoefficient ( )
focusZ ( )



Initialization code:

$("#parallaxID").parallax({
    width               : 840,
    height              : 280,
    enableMouse         : true,
    activateOnClick     : false,
    sensitivityX        : 0.5,  
    sensitivityY        : 0.4,
    easingCoefficient   : 7,
    autoZCoordinate     : true, 
    useCustomZ          : true,
    focusZ              : 0
});
