Introduction

Parallax is a small (5 kb minified & gzipped) jQuery Plugin that helps you to apply parallax effect to any of your content. It has many useful features including:

  • parallax depth (focus) adjustments
  • flexible easing option
  • various mouse interaction options
  • external control support
  • Chaining and playback

If you like parallax - rate it :)

Getting started

To use the parallax plugin, include the jQuery library and the parallax plugin inside the <head> tag of your HTML document:

<script type="text/javascript" src="/path/to/jquery-latest.js"></script> 
<script type="text/javascript" src="/path/to/jquery.real-parallax.js"></script>

Use the following format inside the <body> tag of your HTML document:

<div id="parallaxID">        
    <div style="left:10%;top:50%;">Layer 1</div>
    <div style="left:30%;top:20%;">Layer 2</div>
    <div style="left:50%;top:30%;">Layer 3</div>
    <div style="left:70%;top:70%;">Layer 4</div>
</div>

Note: you can provide top and left position not only in % but also in px


Start parallax when the document is loaded:

$(document).ready(function() 
    { 
        $("#parallaxID").parallax({width: 840,  height: 280});
    } 
);

All together: simplest html file (show / hide)

Configuration

Parallax has many options you can pass in at initialization to achieve different effects:

<script type="text/javascript">
    $(document).ready(function() 
        { 
            $("#parallaxID").parallax({
                width               : 840,
                height              : 280,
                enableMouse         : true,
                activateOnClick     : false,
                sensitivityX        : 0.5,  
                sensitivityY        : 0.4,
                easingCoefficient   : 7,
                autoZCoordinate     : true, 
                useCustomZ          : true,
                focusZ              : 0
            });
        } 
    );
</script>

Table of properties with description:

Property Description
width Parallax width (in pixels), in general, defines the DIV size and mouse-sensitive area. If not provided, width of the parallax DIV will be used.
height Parallax height (in pixels), in general, defines the DIV size mouse-sensitive area. If not provided, height of the parallax DIV will be used.
enableMouse Default value: true. To disable any mouse interactions set it to false: 
$("#parallaxID").parallax({width: 840,  height: 280, enableMouse: false});
activateOnClick Default value: false. In use only if enableMouse=true. Indicates whether activate parallax effect on mouse move (false) or click (true): 
$("#parallaxID").parallax({width: 840,  height: 280, activateOnClick: true});
sensitivityX Default value: 0.5. Sensitivity in horizontal direction. Can be positive, negative (to invert mouse in this direction) or euqal to 0 (zero). In case its set to zero - parallax will not react on mouse movement in horizontal direction: 
$("#parallaxID").parallax({width: 840,  height: 280, sensitivityX: 0});
sensitivityY Default value: 0.4. Sensitivity in vertical direction. Can be positive, negative (to invert mouse in this direction) or euqal to 0 (zero). In case its set to zero - parallax will not react on mouse movement in vertical direction: 
$("#parallaxID").parallax({width: 840,  height: 280, sensitivityY: 0});
easingCoefficient Default value: 7. Can be positive or euqal to 0 (no easing effect). Bigger values make easing effect more prominent: 
$("#parallaxID").parallax({width: 840,  height: 280, easingCoefficient: 15});
useCustomZ Default value: true. By default parallax plugin automatically assign Z coordinate (between -100 and 100) for each layer . Howether, Z-coordinate of each layer can be defined manually using "data-z" (see example). Also layer can be locked (no parallax effect will be applied to this layer) with the help of "data-lock" attribute. Example: 
<div id="parallaxID">        
    <div style="left:10%;top:50%;" data-z="70">Layer 1</div>
    <div style="left:30%;top:20%;">Layer 2</div>
    <div style="left:50%;top:30%;" data-z="50">Layer 3</div>
    <div style="left:70%;top:70%;" data-lock="yes">Layer 4</div>
</div>
If useCustomZ=false then manually provided z-coordinates (see example above) will be ignored. For layers there z-coordinate is no set manually - z-coordinate will be automatically assigned by plugin.
autoZCoordinate Default value: true. By default parallax plugin automatically assigns Z coordinate (between -100 and 100) for each layer. If you want to prevent it, set autoZCoordinate=false (not recommended, use manually provided z-coordinates instead).
focusZ Default value: 0. Determines the Z-coordinate at which no movement accurs. Layers with z<focusZ and z>focusZ move in opposite directions. Layers with z=focusZ doesnt move. To change focusZ: 
$("#parallaxID").parallax({width: 840,  height: 280, focusZ: 60});

Methods

Parallax plugin has a number of methods, especially useful then you want to change parameters or affect paralax at any time directly from your HTML page. Here are all the methods available with examples on how to execute them from your HTML page:

// to change properties of parallax
$('#parallaxID').parallax('option', 'enableMouse', true);           //enable mouse
$('#parallaxID').parallax('option', 'activateOnClick', true);       //activate parallax effect on mouse click
$('#parallaxID').parallax('option', 'sensitivityX', 0.5);           //sensitivity of the mouse in horizontal direction
$('#parallaxID').parallax('option', 'sensitivityY', -0.4);          //negative values invert mouse
$('#parallaxID').parallax('option', 'easingCoefficient', 15);       //bigger value - slower/smoother movement
$('#parallaxID').parallax('option', 'autoZCoordinate', true);       //if true - Z-coordinates will be assigned automatically (between +100 and -100)
$('#parallaxID').parallax('option', 'useCustomZ', true);            //if false - Z-coordinates will be assigned automatically (between +100 and -100)
$('#parallaxID').parallax('option', 'focusZ', 50);                  //change the focus of the parallax


// to emulate mouse movements ant to interact with parallax
$('#parallaxID').parallax('mouseTo', {x:0, y:0});                                   //emulate mouse movement to x, y in px
$('#parallaxID').parallax('mouseToRatio', {x:0.7, y:0.9});                          //emulate mouse movement to x, y in provided as ratio of width, height 
$('#parallaxID').parallax('mouseToRandom');                                         //emulate mouse movement random position
$('#parallaxID').parallax('reset');                                                 //emulate mouse movement to the center
$('#parallaxID').parallax('pause', 3000);                                           //pause parallax effect for 3 seconds
$('#parallaxID').parallax('unpause');                                               //unpause parallax effect
$('#parallaxID').parallax('triggerPause');                                          //pause/unpause parallax effect
$('#parallaxID').parallax('centerTo', {id:'layerID', align:'tl', offx:0, offy:0});  //apply parallax effect to make layers top-left ('tl') corner appear in center with given offsets. Use letters t(top), b(bottom), l(left) and r(right) to align.
$('#parallaxID').parallax('layer', 'layerID', {x:200, y:100});                      //move layer layerID to position x, y in px



//CHAINS AND PLAYBACK

//add all above actions to chain to execute them later
$('#parallaxID').parallax('addToChain','mouseTo', {x:0, y:0});
$('#parallaxID').parallax('addToChain','mouseToRatio', {x:0.7, y:0.9});
$('#parallaxID').parallax('addToChain','mouseToRandom');
$('#parallaxID').parallax('addToChain','reset');
$('#parallaxID').parallax('addToChain','centerTo', {id:'layerID', align:'tl', offx:0, offy:0});
$('#parallaxID').parallax('addToChain','pause', 3000);
$('#parallaxID').parallax('addToChain','triggerPause');
$('#parallaxID').parallax('addToChain','unpause');  


//playback, use also pause (see above) to pause the chain
$('#parallaxID').parallax('startChain', true);                  //play in loop, change parameter to false to play the chain only once
$('#parallaxID').parallax('stopChain');                         //stops chain playback
$('#parallaxID').parallax('clearChain');                        //clear chain of actions


//CROSS-LINKING TWO PARALLAXES
$('#parallaxID').parallax('linkWith','parallaxToControlID');    //all mouse movements from parallaxID will be translated to parallaxToControlID
$('#parallaxID').parallax('resetLinks');                        //remove all links for parallaxID
