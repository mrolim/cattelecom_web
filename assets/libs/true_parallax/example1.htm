Constructing simple parallax

Below you will find working example(s) and complete, ready to use HTML code for each example.

Remember: always include the correct path to jQuery library and the parallax plugin:

 <!-- PROVIDE THE CORRECT PATH TO .js FILES -->
<script type="text/javascript" src="/path/to/jquery-latest.js"></script> 
<script type="text/javascript" src="/path/to/jquery.real-parallax.js"></script>

Layers positioning

Layer 1 (left:300px;top:30%;)
Layer 2 (left:300px;top:40%;)
Layer 3 (left:300px;top:50%;)
Layer 4 (left:300px;top:60%;)
Layer 5 (left:300px;top:70%;)

HTML code

Complete html code of example above (show / hide)


Note: you can provide top and left position not only in % but also in px

<!DOCTYPE html>
<html xmlns="http://www.w3.org/1999/xhtml">
    <head>
        <meta http-equiv="Content-Type" content="text/html; charset=UTF-8" />
        <meta name="description" content="Real Parallax effect jQuery plugin" />
        <title>Parallax</title>
        
    <style type="text/css">
    #parallaxID {
        position: relative;
        overflow:hidden;
        padding: 0px;   
        margin: 0px;
        border:1px solid #ccc;          
    }
    </style>
        
    <!-- PROVIDE THE CORRECT PATH TO .js FILES -->
        <script type="text/javascript" src="/path/to/jquery-latest.js"></script> 
        <script type="text/javascript" src="/path/to/jquery.real-parallax.js"></script> 
        
        <script type="text/javascript">
            $(document).ready(function() 
                { 
                    $("#parallaxID").parallax({width: 640,  height: 280});
                } 
            );
        </script>   
        
    </head>

    <body>    
        <div id="parallaxID">        
            <div style="left:300px;top:30%;">Layer 1</div>
            <div style="left:300px;top:40%;">Layer 2</div>
            <div style="left:300px;top:50%;">Layer 3</div>
            <div style="left:300px;top:60%;">Layer 4</div>
            <div style="left:300px;top:70%;">Layer 5</div>
        </div>
    </body>
</html>

Z-coordinates

By default parallax plugin can assign Z-coordinates automatically (between z=-100 and z=+100) like in example above. Howether, for fine tuning z-coordinates can be adjusted manually. Layers can also be locked at place.

Layer 1 (z:auto)
Layer 2 (z:20)
Layer 3 (z:30)
Layer 4 (locked)
Layer 5 (z=-150)

HTML code

Complete html code of example above (show / hide)


<!DOCTYPE html>
<html xmlns="http://www.w3.org/1999/xhtml">
    <head>
        <meta http-equiv="Content-Type" content="text/html; charset=UTF-8" />
        <meta name="description" content="Real Parallax effect jQuery plugin" />
        <title>Parallax</title>
        
        <style type="text/css">
        #parallaxID {
            position: relative;
            overflow:hidden;
            padding: 0px;   
            margin: 0px;
            border:1px solid #ccc;          
        }
        </style>
        
        <!-- PROVIDE THE CORRECT PATH TO .js FILES -->
        <script type="text/javascript" src="/path/to/jquery-latest.js"></script> 
        <script type="text/javascript" src="/path/to/jquery.real-parallax.js"></script> 
        
        <script type="text/javascript">
            $(document).ready(function() 
                { 
                    $("#parallaxID").parallax({width: 640,  height: 280});
                } 
            );
        </script>   
        
    </head>
    
    <body>    
        <div id="parallaxID">        
           <div style="left:300px;top:30%;">Layer 1 (z:auto)</div>
                <div style="left:300px;top:40%;" data-z="20">Layer 2 (z:20)</div>
                <div style="left:300px;top:50%;" data-z="30">Layer 3 (z:30)</div>
                <div style="left:300px;top:60%;" data-lock="yes">Layer 4 (locked)</div>
                <div style="left:300px;top:70%;" data-z="-150">Layer 5 (z=-150)</div>
            </div>
    </body>
</html>
