Cross-linking

Below you will find working example(s) and complete, ready to use HTML code for each example.

Remember: always include the correct path to jQuery library and the parallax plugin:

 <!-- PROVIDE THE CORRECT PATH TO .js FILES -->
<script type="text/javascript" src="/path/to/jquery-latest.js"></script> 
<script type="text/javascript" src="/path/to/jquery.real-parallax.js"></script>

Link two paralaxes

It is possible to link parallaxes to each other. Even if they have different properties.

Im the MASTER!:)...

SensitivityY>0

SensitivityX high

Focus on the SUN

...tell me what to do my master

SensitivityY<0

SensitivityX low

Focus behind the SUN

Code to link to parallaxes:

//link two parallaxes
$('#masterparallax').parallax('linkWith','slaveparallax');

HTML code

Complete html code of example above (show / hide)

<!DOCTYPE html>
	<html xmlns="http://www.w3.org/1999/xhtml">
		<head>
			<meta http-equiv="Content-Type" content="text/html; charset=UTF-8" />
			<meta name="description" content="Real Parallax effect jQuery plugin" />
			<title>Parallax</title>
			
			<style type="text/css">
			#masterparallax, #slaveparallax {
				position: relative;
				overflow:hidden;
				padding: 0px;   
				margin: 0px;
				border:1px solid #ccc;          
			}
			</style>
			
			<!-- PROVIDE THE CORRECT PATH TO .js FILES -->
			<script type="text/javascript" src="/path/to/jquery-latest.js"></script> 
			<script type="text/javascript" src="/path/to/jquery.real-parallax.js"></script> 
				
			<script type="text/javascript">
				$(document).ready(function() 
					{ 
						$("#masterparallax").parallax({width: 450,  height: 280, sensitivityX:1});
						$("#slaveparallax").parallax({width: 330,  height: 280, sensitivityY:-0.4, sensitivityX:0.5, focusZ:40});
						
						//link two parallaxes
	                    $('#masterparallax').parallax('linkWith','slaveparallax');
					} 
				);
			</script>   
			
		</head>
		
		<body>   
		
			<div id="masterparallax" class="box">        
				<div style="left:5px;top:5px;" data-lock="yes"><h3>Im the MASTER!:)...</h3></div>
				<div style="left:50%;top:20%;" data-z="70"><img src="img/planets/planet1.png"/></div>
				<div style="left:40%;top:30%;" data-z="0"><img src="img/planets/sun.png"/></div>
				<div style="left:25%;top:30%;" data-z="-70"><img src="img/planets/planet2.png"/></div>
			</div>
			
			<div id="slaveparallax" class="box">        
				<div style="left:5px;top:5px;" data-lock="yes"><h3>...tell me what to do my master</h3></div>
				<div style="left:50%;top:20%;" data-z="70"><img src="img/planets/planet1.png"/></div>
				<div style="left:40%;top:30%;" data-z="0"><img src="img/planets/sun.png"/></div>
				<div style="left:25%;top:30%;" data-z="-70"><img src="img/planets/planet2.png"/></div>
			</div>
			
		</body>
	</html>

Link many paralaxes

Its possible to cross-link as many parallaxes as you like.

master

GOD like parallax
mouse over here

slave

HTML code

Complete html code of example above (show / hide)

<!DOCTYPE html>
<html xmlns="http://www.w3.org/1999/xhtml">
    <head>
        <meta http-equiv="Content-Type" content="text/html; charset=UTF-8" />
        <meta name="description" content="Real Parallax effect jQuery plugin" />
        <title>Parallax</title>
        
        <style type="text/css">
        .box{
            position: relative;
            overflow:hidden;
            padding: 0px;   
            margin: 0px;
            border:1px solid #ccc;          
        }
        </style>
        
        <!-- PROVIDE THE CORRECT PATH TO .js FILES -->
        <script type="text/javascript" src="/path/to/jquery-latest.js"></script> 
        <script type="text/javascript" src="/path/to/jquery.real-parallax.js"></script> 
            
        <script type="text/javascript">
            $(document).ready(function() 
                { 
                    $("#parallax1").parallax({width: 300,  height: 280, sensitivityY:0.4, sensitivityX:0.5, focusZ:-5});
                    $("#parallax2").parallax({width: 300,  height: 280, sensitivityY:-0.4, sensitivityX:0.5, focusZ:-5});
                    $('#controlBox').parallax({width: 150,  height: 280,focusZ:30, easingCoefficient:0}); 
                
                    //link parallaxes
                    $('#controlBox').parallax('linkWith','parallax1');
                    $('#controlBox').parallax('linkWith','parallax2');
                    $('#parallax1').parallax('linkWith','parallax2');
                } 
            );
        </script>   
        
    </head>
    
    <body>   
    
        
        <div id="parallax1" class="box">        
            <div style="left:5px;top:5px;" data-lock="yes"><h3>master</h3></div>
            <div style="left:50%;top:10%;" data-z="70"><img src="img/planets/planet2.png"/></div>
            <div style="left:40%;top:37%;" data-z="0"><img src="img/planets/sun.png"/></div>
            <div style="left:25%;top:50%;" data-z="-70"><img src="img/planets/planet1.png"/></div>
        </div>
    
        <div id="controlBox">
            <div style="left:15px;top:40px;" data-lock="yes">GOD like parallax</div>                
            <div style="left:20px;top:190px;" data-lock="yes">mouse over here</div>             
            <div style="left:15%;top:30%;" data-z="-45"><img src="img/arrows.png" /></div>              
        </div>
    
        <div id="parallax2" class="box">        
            <div style="left:5px;top:5px;" data-lock="yes"><h3>slave</h3></div>
            <div style="left:50%;top:10%;" data-z="70"><img src="img/planets/planet2.png"/></div>
            <div style="left:40%;top:37%;" data-z="0"><img src="img/planets/sun.png"/></div>
            <div style="left:25%;top:50%;" data-z="-70"><img src="img/planets/planet1.png"/></div>
        </div>
        
    
    </body>
</html>
