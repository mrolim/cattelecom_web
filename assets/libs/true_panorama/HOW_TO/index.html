Introduction

Real Panorama is a small jQuery Plugin that helps you to apply moving panorama effect to any of your images. It has many useful features including:

  • Fullscreen, fullwidth or fixed size modes
  • 100% Responsive
  • Mouse controllable
  • Touchscreen support

Pattern overlay

To overlay panorama with the pattern use background-image css style.

#mypanorama {
	width:350px;
	background-image:url('path/to/pattern.png');
}
Note: if you want to increase rendering performance of some very old browsers (like IE7) use big sized pattern images.

Settings

Panorama has sattings you can pass in at initialization to achieve different effects.

show / hide

$(document).ready(function(){

	//START panorama
	$('#mypanorama').panorama({

		//all settings are optional, you can delete them or provide your values (default values are shown here)
	
		size		: false,	//See 'Setting the size of the panorama' section			
		speed		: 20,		//target speed px/sec (if negative - move to opposite direction on start, if zero - not moving at start)
		move		: 200,		//max speed px/sec on mouse move (if negative - inverted sensitivity, if zero - not sensitive)
		wheel		: 1,		//sensitivity on mouse wheel (if negative - inverted sensitivity, if zero - not sensitive)					
		drag		: 1,		//sensitivity on drag (if negative - inverted sensitivity, if zero - not sensitive)			
		gravity		: 200,		//max speed px/sec on mobile device movement (if negative - inverted sensitivity, if zero - not sensitive. In use only if mobile device hardware supports this feature)			
		attenuation	: 4,		//attenuation koefficient (not negative number)
		fade		: 300,		//fade on image load in milliseconds (set to zero to remove fade effect. Fade is forced to be zero on mobile devices)
		fps			: 60		//default fps limit
		
	}); 

});

Setting the size of the panorama

The correct way to set the WIDTH of the panorama is using CSS

#mypanorama {width:350px;}

If you want to make it responsive (or WULLWIDTH) - use percents instead of pixels, e.g.

#mypanorama {width:100%;}

To set HEIGHT of the panorama use "size" setting. For FIXED size of panorama use e.g. size = '100px' then height will be equal to 100 pixels (and panorama image will be scaled to fit 100px height). 

$(document).ready(function(){ 
	$("#mypanorama").panorama({
		size : '100px',   // NOTE: dont forget to use 'px' at the end, e.g. '100px' or '150px'
	});  
});

To set fixed WIDTH/HEIGHT ratio of the panorama set "size" setting to number, e.g. if size = 1 then height will be the same as width. Or if size = 2 then height will be always = width/2 (twice as less then width). Always! even if width is given in %. 

$(document).ready(function(){ 
	$("#mypanorama").panorama({
		size : 2,   // NOTE: without 'px' at the end (in contrast to preveous example) size represent width/height ratio !!!
	});  
});

To set panorama in FULLSCREEN mode set size = 'fullscreen' (or true) 

$(document).ready(function(){ 
	$("#mypanorama").panorama({
		size : true,   // set 'fullscreen' or true
	});  
});

Note: if you do NOT provide "size" parameter or set size=false - the height of the image will be used.

Methods

This part is for advanced users only. If you dont know how to use methods - you dont need to reed this section. Tags panorama plugin has a number of methods, especially useful then you want to change parameters or affect panorama at any time directly from your HTML page. Here are all the methods available with examples on how to execute them from your HTML page:

show / hide

// to change properties of panorama
$('#mypanorama').panorama('option', 'speed', -200);     //set movement speed to -200px/sec
$('#mypanorama').panorama('option', 'move', 400);     //set maximal speed on mouse move to 400px/sec (if zero - mouse nonsensitive)
$('#mypanorama').panorama('option', 'gravity', 200);     //set maximal speed on mobile device movement to 200px/sec (if zero - non-sensitive)
$('#mypanorama').panorama('option', 'wheel', 2);     //sensitivity on mouse wheel (if zero - mouse wheel nonsensitive)
$('#mypanorama').panorama('option', 'drag', 1);     //sensitivity on drag (if zero - non-draggable)
$('#mypanorama').panorama('option', 'attenuation', 4);     //attenuation koefficient (if zero - non-attenuating)
$('#mypanorama').panorama('option', 'fade', 800);     //image fade on load in milliseconds(if zero - no fade effect)
$('#mypanorama').panorama('option', 'fps', 60);     //target frames per second (60 or less - reccomended)


// to emulate mouse movements and to interact with panorama
$('#mypanorama').panorama('play');					    //start panorama movement speed defined in settings
$('#mypanorama').panorama('play', 200);					//start panorama movement with 200px/sec
$('#mypanorama').panorama('pause');					    //pause panorama	
$('#mypanorama').panorama('pause', 3000);				//pause panorama for 3 seconds

FAQ

Q: My panorama image doesn't show up. Why?

A: Some browsers (particularly mobile browsers) may have the limitations on maximal image dimentions. Make sure that your panorama image is as small as possible:
1) If your image is 6000px height it makes sence to resize it to e.g. 1000px height which is enought to completely fit into almost any screen size/resolution.
2) If you want to show panorama with e.g. height=100px its always good to resize the big image to final height (100px).


Q: My panorama image loads too long. Why?

A: The size of your image could be too big (up to several Mb).
1) Reduce the size of the image (1000px or less in height and 72 DPI will be fine).
2) If you want to show panorama with some certain height (see settings "size") its always good to resize the big image to final height.
