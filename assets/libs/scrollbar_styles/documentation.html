Documentation - CSS3 Scrollbar

This documentation will help you in the processing to include CSS3 Scrollbar

1. Installation

1.1. Insert the style-blue.css or other css-color file in the head-tag of your html file. 
    
        <!DOCTYPE html>
        <html>
            <head>
            	
                <link rel="stylesheet" href="style-blue.css" type="text/css" >
                
            </head>
            <body>
            </body>
        </html>
1.3. So you can see the css Scrollbar on the live page.

2. Scrollbar

2.1. You can edit the Scrollbar -webkit-scrollbar. 
    
       	.scrollbar-1::-webkit-scrollbar
        {
        width:13px;height:13px;
        background:transparent
        }

3. Scrollbar Track

3.1. You can edit the Scrollbar Track -webkit-scrollbar-track. 
    
       	.scrollbar-1::-webkit-scrollbar-track
        {
        background:#6fa1d4
        }

4. Scrollbar Corner

4.1. You can edit the Scrollbar Track -webkit-scrollbar-corner. 
    
       	.scrollbar-1::-webkit-scrollbar-corner
        {
        background:#6fa1d4
        }

5. Scrollbar Button

5.1. You can edit the Scrollbar Track -webkit-scrollbar-button. 
    
       	.scrollbar-1::-webkit-scrollbar-button
        {
        width:13px;height:13px
        }

6. Scrollbar Button

6.1. You can edit the Scrollbar Track -webkit-scrollbar-button. 
    
       	.scrollbar-1::-webkit-scrollbar-button
        {
        width:13px;height:13px
        }
6.1. Using own icons. 
    
       	.scrollbar-1::-webkit-scrollbar-button:horizontal:decrement {background:#6fa1d4 url("icons/horizontal-decrement-arrow.gif") center center no-repeat}
        .scrollbar-1::-webkit-scrollbar-button:horizontal:increment {background:#6fa1d4 url("icons/horizontal-increment-arrow.gif") center center no-repeat}
        .scrollbar-1::-webkit-scrollbar-button:vertical:decrement {background:#6fa1d4 url("icons/vertical-decrement-arrow.gif") center center no-repeat}
        .scrollbar-1::-webkit-scrollbar-button:vertical:increment {background:#6fa1d4 url("icons/vertical-increment-arrow.gif") center center no-repeat}
