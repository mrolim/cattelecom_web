Documentation - Page Scrolling

This documentation will help you in the processing to include Page Scrolling

1. Installation

1.1. Insert the style.css and jquery.min.js file in the head-tag of your html file. 
    
        <!DOCTYPE html>
        <html>
            <head>
            	
                <link rel="stylesheet" href="style.css" type="text/css" >
                <script src="jquery.min.js"></script>
                
            </head>
            <body>
            </body>
        </html>
1.2. Insert the html code for pricing table into the body-tag of your html file. 
    
        <body>
        	
            <div class="scrollmenu style1 black">
                <div class="scrollmenu-inner">
                    <div class="scrollmenu-inner-top">
                        <div class="scroll-button stop-1 scroll-top slide-top"> </div>
                    </div>
                    <div class="scrollmenu-inner-bottom">
                        <div class="scroll-button sbottom-1 scroll-bottom slide-bottom"> </div>
                    </div>
                </div>
            </div>
            
        </body>
1.3. So you can see the Page Scrolling Menu on the live page.

5. Scrollmenu - Buttons is show all the time

5.1. You can show the Page Scrolling Menu all the time. 
    
		
        <script type="text/javascript">
            // Scroll to Top
            $(function () {$(window).scroll(function () {});
            $('.slide-top').click(function () { $('body,html').animate({ scrollTop: 0 }, 500); return false;});
            });
            // Scroll to Bottom
            $(function () {$(window).scroll(function () {});
            $('.slide-bottom').click(function () { $('body,html').animate({ scrollTop: $(document).height() - $(window).height() }, 500); return false;});
            });
        </script>
    	
        <div class="scrollmenu style1 black topleft">
            <div class="scrollmenu-inner">
                <div class="scrollmenu-inner-top">
                    <div class="scroll-button stop-1 scroll-top slide-top"> </div>
                </div>
                <div class="scrollmenu-inner-bottom">
                    <div class="scroll-button sbottom-1 scroll-bottom slide-bottom"> </div>
                </div>
            </div>
        </div>

6. Scrollmenu - Buttons fade in and out

6.1. You can fade in and out the Page Scrolling Menu. 
    
		
        <script type="text/javascript">
            // Scroll to Top
            $(function () {$(window).scroll(function () {if ($(this).scrollTop() > 100) $('.slide-top-fade').fadeIn();else $('.slide-top-fade').fadeOut();});
            $('.slide-top-fade').click(function () { $('body,html').animate({ scrollTop: 0 }, 500); return false;});
            });
            // Scroll to Bottom
            $(function () {$(window).scroll(function () {if ($(document).height() - $(window).height() == $(this).scrollTop()) $('.slide-bottom-fade').fadeOut();else $('.slide-bottom-fade').fadeIn();});
            $('.slide-bottom-fade').click(function () { $('body,html').animate({ scrollTop: $(document).height() - $(window).height() }, 500); return false;});
            });
        </script>
    	
        <div class="scrollmenu style1 black topleft">
            <div class="scrollmenu-inner">
                <div class="scrollmenu-inner-top">
                    <div class="scroll-button stop-1 scroll-top slide-top-fade"> </div>
                </div>
                <div class="scrollmenu-inner-bottom">
                    <div class="scroll-button sbottom-1 scroll-bottom slide-bottom-fade"> </div>
                </div>
            </div>
        </div>
